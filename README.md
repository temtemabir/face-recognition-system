# 😊 Face Recognition System

## Overview

Face Recognition System is a computer vision application developed using Python. The system detects and recognizes human faces from images or a live webcam stream using image processing and face recognition techniques.

This project demonstrates the integration of computer vision and machine learning concepts for real-time face identification.

---

## Features

- Face detection
- Face recognition
- Real-time webcam support
- Image processing
- Fast and accurate recognition

---

## Technologies Used

- Python
- OpenCV
- NumPy
- Face Recognition Library
- Computer Vision

---

## Project Objectives

- Detect human faces from images and video.
- Recognize known individuals.
- Apply computer vision techniques using Python.
- Explore practical applications of face recognition technology.
